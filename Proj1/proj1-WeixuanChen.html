By Bayes Theorem,p(f2|d) = p(d|f2)*p(f2)/p(d). p(d|f2) = 0.7, p(f2) = 0.6. And by the law of total probability, p(d) = p(d|f2)p(f2) + p(d|not f2)p(not f2) = 0.7 x 0.6 + 0.1 x 0.4, so we can finalize the result in the code below.
